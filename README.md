# 📚 Exploratory Data Analysis of Amazon Top 50 Best Selling Books

## 🌟 Overview
This project provides a comprehensive exploratory data analysis of the top 50 best-selling books on Amazon. The dataset contains various attributes of the books including titles, authors, genres, and ratings.

## 📊 Data
- **Source:** Amazon
- **Data Size:** Top 50 best-selling books
- **File Format:** CSV

## 🔍 Analysis
This section outlines the analysis performed on the dataset:
- Distribution of genres
- Trends in book ratings
- Most common authors among best sellers

## 📈 Visualizations
The project includes various visualizations to showcase insights from the data such as:
- Bar plots for genre distribution
- Line charts for rating trends
- Pie charts for author contributions

## 💡 Insights
Key findings from the analysis include:
- The most popular genres among the top sellers
- Insights into the correlation between ratings and number of reviews

## 🛠 Requirements
- Python 3.6+
- Pandas
- Matplotlib
- Seaborn

## 💻 Installation
To run this project, clone this repository and install the requirements:
```bash
git clone https://github.com/abinaya-345/Exploratory-Data-Analysis-of-Amazon-Top-50-Best-Selling-Books.git
cd Exploratory-Data-Analysis-of-Amazon-Top-50-Best-Selling-Books
pip install -r requirements.txt
```

## 📜 Usage
To execute the analysis, run the following command:
```bash
python analyze_books.py
```
